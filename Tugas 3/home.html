Tentang Saya

Saya Muhammad Yanwar, Lulusan SMK Rekayasa Perangkat Lunak (RPL) dengan prestasi sebagai siswa terbaik pada Uji Kompetensi Keahlian (UKK) dengan judul aplikasi buku tamu sekolah berbasis web. Saya memiliki pengalaman magang di FEMA IPB University sebagai Arsiparis bagian input data, di mana saya terbiasa bekerja dengan pengelolaan dokumen, pendataan arsip, serta memastikan ketelitian dalam proses input dan administrasi data. Saat ini saya melanjutkan studi di Program Studi Teknik Informatika di STT Terpadu Nurul Fikri. Selama perkuliahan, saya mempelajari dasar-dasar pemrograman, desain ui ux, basis data, jaringan komputer, dan pengembangan aplikasi web. Saya memiliki ketertarikan dalam bidang pengembangan sistem informasi dan berkomitmen untuk terus mengasah kemampuan di dunia teknologi.
